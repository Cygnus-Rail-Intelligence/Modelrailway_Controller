# 🚀 GITHUB UPLOAD ANLEITUNG

## Schritt-für-Schritt: Projekt auf GitHub veröffentlichen

**Dauer:** ~10 Minuten  
**Kosten:** Kostenlos ✅  
**Vorkenntnisse:** Keine erforderlich

---

## ✅ VORAUSSETZUNGEN

### 1. GitHub-Account erstellen

1. Gehe zu https://github.com/signup
2. E-Mail, Passwort, Benutzername eingeben
3. E-Mail bestätigen
4. **Kostenlos!** ✅

### 2. Methode wählen

**OPTION A:** Web-Interface (am einfachsten!)  
**OPTION B:** GitHub Desktop (komfortabel)  
**OPTION C:** Git Kommandozeile (für Profis)

---

## 📦 OPTION A: WEB-INTERFACE (Empfohlen!)

### Schritt 1: Repository erstellen

1. Auf GitHub einloggen
2. Rechts oben **"+"** → **"New repository"**
3. Name: `Cygnus-Rail-Intelligence`
4. Description: `Arduino Cygnus-Rail Controller mit TFT-Display`
5. **Public** auswählen (kostenlos!)
6. README NICHT ankreuzen (haben wir schon!)
7. **"Create repository"**

### Schritt 2: Dateien hochladen

1. **"uploading an existing file"** klicken
2. Alle Dateien rein ziehen (Drag & Drop)
3. Commit: `Initial release v2.3b`
4. **"Commit changes"**

### Schritt 3: Ordner erstellen

1. **"Add file"** → **"Create new file"**
2. Name: `docs/README.md`
3. Inhalt: `# Dokumentation`
4. **"Commit"**
5. Dokumentations-Dateien in docs/ hochladen

### Fertig! 🎉
