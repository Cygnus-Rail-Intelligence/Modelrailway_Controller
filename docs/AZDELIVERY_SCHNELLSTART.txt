## 📌 PIN-VERBINDUNG (AZDelivery Beschriftung):

```
AZDelivery Pin  →  Arduino Mega Pin
─────────────────────────────────────
GND             →  GND
VCC             →  5V
SCL             →  Pin 52 (SCLK)
SDA             →  Pin 51 (MOSI)
RES             →  Pin 49 (RST)
DC              →  Pin 47
CS              →  Pin 48
BLK             →  5V (Backlight)
```

**Wichtig:** AZDelivery nutzt andere Bezeichnungen:
- SCL = Clock
- SDA = Data
- RES = Reset
- BLK = Backlight
