## 📋 VERKABELUNG - Schritt für Schritt:

### SCHRITT 1: Stromversorgung (zuerst!)

```
Display Pin     Kabel-Farbe    Arduino Pin
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
VCC         →   [ROT]      →   5V
GND         →   [SCHWARZ]  →   GND
```

**Prüfen:** Display sollte jetzt Strom haben (evtl. leuchtet LED leicht)

✅ VCC verbunden
✅ GND verbunden

---

### SCHRITT 2: Backlight (Hintergrundbeleuchtung)

```
Display Pin     Kabel-Farbe    Arduino Pin
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
LED         →   [GELB]     →   5V
```

**Prüfen:** Display sollte jetzt hell leuchten (blau/weiß)

✅ LED verbunden

---

### SCHRITT 3: SPI-Kommunikation

```
Display Pin     Kabel-Farbe    Arduino Pin    Beschreibung
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
SCK         →   [GRÜN]     →   Pin 52      (SPI Clock)
SDA         →   [BLAU]     →   Pin 51      (SPI Data)
```

**Wichtig:** Mega 2560 hat feste SPI-Pins!
- Pin 52 = SCK (Clock)
- Pin 51 = MOSI (Data)

✅ SCK an Pin 52
✅ SDA an Pin 51

---

### SCHRITT 4: Steuer-Pins

```
Display Pin     Kabel-Farbe    Arduino Pin    Beschreibung
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
A0          →   [ORANGE]   →   Pin 47      (Data/Command)
Reset       →   [LILA]     →   Pin 49      (Reset)
CS          →   [WEISS]    →   Pin 48      (Chip Select)
```

**⚠️ WICHTIG:** A0 ist NICHT Analog Pin A0!
A0 bedeutet hier "Address 0" = Data/Command Pin

✅ A0 an Pin 47 (DC)
✅ Reset an Pin 49
✅ CS an Pin 48

- Display leuchtet hell (blau/weiß) ✓

### Test 3: Kommunikation

- Display zeigt "CYGNUS-RAIL" ✓

## ❌ HÄUFIGE FEHLER:

### Fehler 1: A0 falsch angeschlossen
```
FALSCH: A0 (Display) → A0 (Arduino)  ❌
RICHTIG: A0 (Display) → Pin 47       ✅
```

### Fehler 2: SPI-Pins verwechselt
```
FALSCH: SCK → Pin 13, SDA → Pin 11   ❌
RICHTIG: SCK → Pin 52, SDA → Pin 51  ✅
```
**Mega 2560 hat andere SPI-Pins als Arduino Uno!**

### Fehler 3: LED nicht angeschlossen
```
Symptom: Display schwarz, aber Programm läuft
Lösung: LED → 5V anschließen
```

### Fehler 4: GND vergessen
```
Symptom: Merkwürdiges Verhalten, Artefakte
Lösung: Gemeinsame Masse wichtig!
```

## 🆘 TROUBLESHOOTING:

### Display komplett schwarz:
→ LED an 5V? Backlight muss leuchten!

### Display leuchtet, aber kein Text:
→ Alle Pins korrekt? Besonders A0 → Pin 47

### Streifen oder Artefakte:
→ GND sicher verbunden?
→ Kabel zu lang? (max. 20cm)

### Komische Farben:
→ Normal! invertDisplay(true) im Code aktiv

### Programm startet nicht:
→ USB-Kabel OK?
→ Richtiger COM-Port?
